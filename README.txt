Grocery List Application

This is a web application to manage a grocery list, developed using AngularJS and Bootstrap on WebStorm.

To start the app open index.html using firefox browser.

How to add a new grocery item?
Click on "Add Grocery Item" to add a new item and then click save.

How to edit grocery item?
click on the pencil button to the right of the desired item and then save.

How to remove a grocery item?
click on the trash button to the right of the item.

How to check mark an item?
click on the green button to the left of the desired item to scratch it from the list.
